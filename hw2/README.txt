For the result with double precision, 

```
g++ -std=c++14 hw2.cpp -o hw2;
```

For the result with single precision, 

```
g++ -std=c++14 axpy.cpp -o axpy;
g++ -std=c++14 gemv.cpp -o gemv;
g++ -std=c++14 gemm.cpp -o gemm;
./axpy > axpy.csv;
./gemv > gemv.csv;
./gemm > gemm.csv;
```
